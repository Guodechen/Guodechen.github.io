今读Effective c++ 01-03条款

01 视C++为一个语言联邦

由于有C所以有C++，并且有了object-oriented-C++，template C++,STL

02 尽量以const，enum，inline替换#define

或许#define不视为语言的一部分，因此编译的时候编译器不会检查是否有错，因此不会记录在符号表内。于是经常容易发生错误。
1） 字符串用const的定义：
const char* const authorName = “Scott Meyers”;
尽量定义为
const std::string authorName(“Scott Meyers”);
2） class中经常使用到的用static的变量尽量用enum代替
class GamePlayer {
static const int NumTurns = 5;
int scores[NumTurns];
…
}
最好使用
class GamePlayer {
enum { NumTurns = 5};

    int scores[NumTurns];
    ...
}

3） 对于形似函数的宏，最好改用inline函数替换#defines

尽可能多的使用const

### 如果关键字const出现在星号的左边，代表被指物是常量；如果出现在星号右边，表示指针自身是常亮；如果两边都有，表示被指物和指针两者都是常量。
1）自身成员函数可以使用
class ...{
    const  ... operator * (const ...& lhs,const ...&rhs);
}
可以避免写代码时a * b = c的形式出现无法估量的错误
2）const 成员函数
class TextBlock {
public:
    ...
    const char&[] operator[](std::size_t position) const  
    {return text[position]; }            //const 对象

    char&[] operator[](std::size_t position)
    {return text[position]; }            //非const 对象
private:
    std::string text;
}
TextBlock tb("Hello");            //调用非const
std::cout << tb[0];
TextBlock ctb("World");            //调用const
std::cout << ctb[0];
只能修改未调用const的函数值，不能调用使用const函数值
3） c++的摆动场mutable（可变的）
当const函数修改private数据成员的值的时候，将其变成可变数据成员
class CTextBlock {
    public:
        ...
        std::size_t length() const;
    private:
        char *pText;
        std::size_t textLength;
        bool lengthIsValid;
};
std::size_t cTextBlock::length() const
{
    if(!lengthIsvalid){
        textLength = std::strlen(pText); //错误！在const成员函数内
        lengthIsValid = true;             // 不能复制给textLength
    }                                     // 和lengthIsValid
    return textLength;
}
应该使用
class CTextBlock {
    public:
        ...
        std::size_t length() const;
    private:
        char *pText;
        mutable std::size_t textLength;
        mutable bool lengthIsValid;
};
std::size_t cTextBlock::length() const
{
    if(!lengthIsvalid){
        textLength = std::strlen(pText); 
        lengthIsValid = true;             
    }                                     
    return textLength;
}
4） 在使用const和none-const成员函数中避免重复（还得继续参透）
